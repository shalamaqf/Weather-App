## Title
Weather App

## Description
A simple weather search application that fetches real-time weather data for any location. The project uses asynchronous JavaScript such as Promises and async/await, also consumes a weather API. User be able to view the location's temparature and toggle between Celcius and Fahrenheit.

## Technologies
- HTML: for structure
- css: for styling
- JavaScript: for logic and dom
- Webpack: for code bundling
- Git: for version control
